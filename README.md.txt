# Luna Cybersecurity Chatbot 🤖🔒

Luna is a friendly, console-based chatbot that helps users learn about cybersecurity topics like phishing, password safety, and more.

## Features
- Sentiment detection with friendly responses
- Random cybersecurity tips
- Personalized user memory
- Natural conversation flow

## How to Run
Open the project in Visual Studio and run `Program.cs`.
